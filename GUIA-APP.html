Como Actualizar la App (Deploy)

Cada vez que cambies codigo, Railway redeploya automaticamente al hacer push a GitHub:

1
Abre terminal en C:\Users\PC\warehouse-web
2
Ejecuta: git add -A && git commit -m "descripcion del cambio"
3
Ejecuta: git push origin main
4
Railway detecta el push y redeploya automaticamente en ~3 minutos
O desde terminal: railway up --service triumphant-growth

Cambiar el Codigo de Acceso al Portal

  • Entra a la app → Perfil → seccion "Portal Access Code" (solo visible para el owner)
  • Escribe el codigo actual y el nuevo → Update
  • Toma efecto inmediatamente, sin reiniciar nada
